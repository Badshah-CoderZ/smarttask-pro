#  SmartTask Pro – AI Productivity Dashboard
by **badshah-coderz**

A full-stack productivity platform designed to manage tasks, track time, set goals, and visualize growth – all inside one dashboard.

---

##  Core Features
-  Task Create/Edit/Delete
-  Pomodoro Countdown Timer
-  Productivity Analytics Dashboard
-  Secure User Login / Auth System
-  Cloud Database Sync (MongoDB)
-  AI Suggestions (Future Release)
-  Fully Responsive

---

##  Tech Stack
Frontend: React.js / Next.js  
Backend: Node.js + Express  
Database: MongoDB  
Auth: JWT
